# Agents.md

## Project Overview

Personal portfolio/developer site for Emil Krautmann (ecoders.dev). Built with TanStack Start (SSR React framework), TypeScript, Tailwind CSS v4, and Shadcn UI components. Deployed to Netlify.

## Tech Stack

- **Framework**: TanStack Start (TanStack Router + Vite)
- **UI**: React 19, Tailwind CSS v4, Shadcn UI (Radix UI primitives), Lucide React icons
- **Language**: TypeScript
- **Testing**: Vitest + Testing Library
- **Linting/Formatting**: ESLint (tanstack/eslint-config), Prettier
- **Deployment**: Netlify

## Key Conventions

- **Path alias**: `#/*` maps to `./src/*` (use `@/` in imports)
- **Routing**: File-based via TanStack Router — add files to `src/routes/` to create routes
- **Components**: Add Shadcn components with `pnpm dlx shadcn@latest add <component>`
- **Styling**: Tailwind CSS v4 utility classes; dark mode via class strategy with `data-theme` attribute
- **Theme**: Dark/light/auto mode persisted in `localStorage` under key `theme`; initialized via inline script in `__root.tsx` to avoid flash

## Notes for Agents

- Do not modify `src/routeTree.gen.ts` — it is auto-generated by TanStack Router
- The `Header` and `Footer` are rendered globally in `__root.tsx`, not per-route
- Prefer `cn()` from `src/lib/utils.ts` for conditional class merging
- Keep demo/placeholder files in `public/` unless explicitly removing them
